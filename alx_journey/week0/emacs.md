# Emacs
## Emacs keybindings
* C: Ctrl
* M: Meta and/or Alt.
## Emacs cmds
* open a file : ``` C-x C-f ```
* save a file : ``` C-x C-s ```
* cut an entire line : ``` C-k ```
* paste the copied or cut text : ``` C-y ```
* search forword :  ``` C-s ```
* undo the last change made :  ``` C-x u ```
* quit emacs :  ``` C-x C-c ```
