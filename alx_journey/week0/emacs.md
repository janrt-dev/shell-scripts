# Emacs
## Emacs keybindings
* C: Ctrl
* M: Meta and/or Alt.
## Emacs cmds
* open a file : ```sh C-x C-f ```
* save a file : ```sh C-x C-s ```
* cut an entire line : ```sh C-k ```
* paste the copied or cut text : ```sh C-y ```
* search forword :  ```sh C-s ```
* undo the last change made :  ```sh C-x u ```
* quit emacs :  ```sh C-x C-c ```
